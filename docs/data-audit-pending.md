## ペンディング項目

### A. 単一 selectionType で登録されているが、実際には両方実施している学部

多くの大学が同一学部で「総合型選抜」と「学校推薦型選抜」の両方を実施している。現在の `Faculty` 型は `selectionType` を 1 つしか持てないため、正確な表現には **entry 分割 (Pattern β)** が必要。

該当例:
- **北海道大学 フロンティア入試**: 大半の学部が Type I (学校推薦型) + Type II (総合型) の両立て
- **九州大学**: 医学部・薬学部・歯学部以外の学部も学校推薦型を設置している場合がある
- **東北大学**: 全学部で AO 入試と学校推薦型選抜が並存
- **京都大学 特色入試**: 薬学部は薬科学科 (推薦) + 薬学科 (総合) で分かれる

スクレイパー対応時に要対応 — 大学公式 PDF/HTML から各入試枠を抽出し、`selectionType` 別に `Faculty` エントリを分割する。
